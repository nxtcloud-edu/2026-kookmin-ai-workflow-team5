# Wiki 갱신 — progress.md 업데이트

작업 완료 후 LLM-Wiki의 progress.md에 오늘 작업 로그를 추가합니다.
인자 없음 — 대화 맥락에서 내용을 추론합니다.

## 실행 순서

1. 오늘 날짜를 `YYYY-MM-DD` 형식으로 확인합니다.
2. 아래 파일을 읽어 오늘 날짜 섹션이 이미 있는지 확인합니다:
   `C:\Users\kyjkm\llm-wiki-vault-template\LLM-Wiki\10-projects\investment-dashboard\progress.md`
3. 이 대화에서 오늘 한 일, 확인한 것, 막힌 점, 남은 일을 정리합니다.
4. progress.md 파일 맨 끝에 새 섹션을 추가합니다.

## 날짜 섹션 처리 규칙

- 오늘 날짜 섹션이 **없으면**: `## YYYY-MM-DD` 새 섹션으로 추가합니다.
- 오늘 날짜 섹션이 **이미 있으면**: `## YYYY-MM-DD (이어서 — [핵심 작업 한 줄])` 형식으로 추가합니다.

## 추가할 블록 형식

대화에서 아래 항목을 추론해서 채웁니다. 해당 항목이 없으면 `- 없음`으로 씁니다.

```
## YYYY-MM-DD

### 오늘 한 일

- [완료된 작업, 동사로 시작하는 한 줄씩]

### 확인한 것

- [발견한 사실, 동작 확인한 것, 알게 된 것]

### 막힌 점

- [해결 못 한 것, 다음 세션으로 넘긴 것]

### 남은 일

- [다음에 해야 할 것, 미완료 항목]

---
```

## 추가 위치

progress.md 파일의 **맨 끝**에 추가합니다. 기존 내용은 건드리지 않습니다.

## 완료 보고

추가한 내용을 그대로 출력하고 `progress.md에 기록했습니다.` 한 줄로 마칩니다.
